# SQLi辞書

Google 日本語入力用のSQLi辞書。

## インストール方法
Google 日本語入力 辞書ツールを開いて新規辞書にインポートを選択してsqli_dict.txtをインポートする。

![image](./screenshot.png)

## 使い方
全角で s[数字]を打つとsqliが変換候補が出る。

- s１: MySQLへのSQLiチェック
- s２: MySQLから情報を抜くやつ
